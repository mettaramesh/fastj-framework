Quick Start

HelloWorld.java 
@FastJApplication
@EnableSwagger
public class HelloWorld {
    
    @Get("/hello")
    @Description("Simple hello world endpoint")
    public String hello(@QueryParam String name) {
        return "Hello, " + (name != null ? name : "World") + "!";
    }
    
    public static void main(String[] args) {
        FastJ.run(HelloWorld.class, args);
    }
}

Users List

UserController.java 
@Path("/api")
@FastJController
public class UserController {
    
    @Inject
    private UserService userService;
    
    @Get("/users")
    @Description("Get all users")
    public List getAllUsers() {
        return userService.findAll();
    }
    
    @Post("/users")
    @Validate
    public CompletableFuture createUser(@RequestBody User user) {
        return CompletableFuture.supplyAsync(() -> 
            userService.create(user)
        );
    }
    
    @Put("/users/{id}")
    public User updateUser(@PathParam String id, @RequestBody User user) {
        return userService.update(Long.parseLong(id), user);
    }
    
    @Delete("/users/{id}")
    public ResponseEntity deleteUser(@PathParam String id) {
        userService.delete(Long.parseLong(id));
        return ResponseEntity.ok("User deleted successfully");
    }
}

Async & Virtual Threads

AsyncController.java 
@FastJController
public class AsyncController {
    
    @Get("/async/users")
    public CompletableFuture> getAsyncUsers() {
        return CompletableFuture
            .supplyAsync(() -> userService.findAll())
            .thenApply(users -> {
                // Heavy processing on virtual thread
                return users.stream()
                    .map(this::enrichUserData)
                    .collect(Collectors.toList());
            });
    }
    
    @Get("/stream/events")
    @Produces(MediaType.TEXT_EVENT_STREAM)
    public Publisher streamEvents() {
        return Multi.createFrom().ticks()
            .every(Duration.ofSeconds(1))
            .map(tick -> "Event " + tick);
    }
}

Getting Started

FastJ is a modern Java web framework that brings the simplicity of FastAPI to the Java ecosystem.

pom.xml 
<dependency>
    <groupId>com.fastj</groupId>
    <artifactId>fastj-core</artifactId>
    <version>1.0.0</version>
</dependency>

@EnableSwagger

    public static void main(String[] args) {

Annotation Reference

Architecture

FastJ is built on modern Java features including virtual threads, dependency injection, and annotation processing.

Key Components

  • Router: Handles HTTP request routing based on annotations
  • DI Container: Manages dependency injection and bean lifecycle
  • Validator: Automatic request validation using annotations
  • Serializer: JSON serialization/deserialization
  • OpenAPI Generator: Automatic API documentation generation
